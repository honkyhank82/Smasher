QUICKSTART - EASIEST METHOD:

1. Open in browser:
   → icon-generator.html

2. Click "Download" under each icon

3. Save files to assets/ folder:
   - icon.png
   - adaptive-icon.png
   - favicon.png

4. Rebuild app:
   npx expo prebuild --clean
   npx expo run:ios    (or run:android)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ALTERNATIVE - AUTOMATED METHOD:

1. Install sharp:
   npm install sharp

2. Run generator:
   node generate-icon.js

3. Rebuild app:
   npx expo prebuild --clean
