# Branding

## Visual Identity

### Assets
- **Application Icon**: `app-rn/assets/icon.png` (256x256, PNG)
  - Adaptive Icon (Android): `app-rn/assets/adaptive-icon.png`
  - Web Favicon: `app-web/icon.png`
- **Application Logo**: `app-rn/assets/logo.png` (High-resolution, PNG)
  - Also used in Web: `app-web/logo.png`
- **Splash Screen**: `app-rn/assets/splash.png` (Optimized for mobile)

### Colors
- **Primary**: Deep Dark Red (check theme configuration)
- **Secondary**: Silver/Chrome (#C0C0C0)
- **Background**: Black (#000000)

### Usage
- The logo should be used on the Login/Auth screens.
- The icon is used for the app launcher and browser tab.
- Splash screen is displayed during app initialization.
